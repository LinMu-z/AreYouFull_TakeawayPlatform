# frontend 前端模块
- build后的前端模块，用于部署前端部分。
- 出于前后端分离的需要，项目结构中该部分需要经过build（npm run build），其目录结构才可被flask识别。
- /dist 包含用到的静态资源，

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
